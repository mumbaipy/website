## Install

### Installation MkDocs

To install MkDocs, run the following command from the command line:

```bash
pip install mkdocs
```

### Installation theme

Install the theme using PIP:

```bash
pip install mkdocs-simple-blog
```

### Preview/ Run Locally


```bash
mkdocs serve
```

## Need Help?

Read the MkDocs [Documentation](https://www.mkdocs.org/getting-started/) :)
